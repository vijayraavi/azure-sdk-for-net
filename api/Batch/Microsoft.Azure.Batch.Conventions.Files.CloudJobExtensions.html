GetOutputStorageContainerUrl(CloudJob, CloudStorageAccount)

Gets the URL, including a Shared Access Signature (SAS) that permits writing, for the job's output storage container in Azure blob storage. This URL is suitable for passing to tasks so they can use the JobOutputStorage(Uri) or TaskOutputStorage(Uri, String) constructors that take a .

Parameters

Returns

System.String

The URL, including SAS, of the job output container.
Remarks

The SAS expires after 7 days. This default is chosen to match the maximum time that tasks can remain active.
