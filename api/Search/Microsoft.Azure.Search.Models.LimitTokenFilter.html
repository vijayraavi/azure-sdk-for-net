Class LimitTokenFilter

Limits the number of tokens while indexing. This token filter is implemented using Apache Lucene. http://lucene.apache.org/core/4_10_3/analyzers-common/org/apache/lucene/analysis/miscellaneous/LimitTokenCountFilter.html

System.String name

The name of the token filter. It must only contain letters, digits, spaces, dashes or underscores, can only start and end with alphanumeric characters, and is limited to 128 characters.
System.Nullable<System.Int32> maxTokenCount

The maximum number of tokens to produce. Default is 1.
System.Nullable<System.Boolean> consumeAllTokens

A value indicating whether all tokens from the input must be consumed even if maxTokenCount is reached. Default is false.
