GetQueues(Nullable<GetQueuesOptions>, CancellationToken)

The GetQueues(Nullable<GetQueuesOptions>, CancellationToken) operation returns an async sequence of queues in the storage account. Enumerating the queues may make multiple requests to the service while fetching all the values. Queue names are returned in lexicographic order.

For more information, see https://docs.microsoft.com/en-us/rest/api/storageservices/list-queues1

GetQueuesAsync(Nullable<GetQueuesOptions>, CancellationToken)

The GetQueuesAsync(Nullable<GetQueuesOptions>, CancellationToken) operation returns an async collection of queues in the storage account. Enumerating the queues may make multiple requests to the service while fetching all the values. Queue names are returned in lexicographic order.

For more information, see https://docs.microsoft.com/en-us/rest/api/storageservices/list-queues1

Remarks

Use an empty marker to start enumeration from the beginning. Queue names are returned in lexicographic order. After getting a segment, process it, and then call ListQueuesSegment again (passing in the next marker) to get the next segment.
